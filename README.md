# 不動産売却 収支計算ツール

ViVi不動産株式会社の売主様向け収支計算ツール(単一HTML・サーバー不要)。

- 査定書PDFを読み込み、お客様名・物件名・所在地・各価格を自動入力(pdf.js)
- 仲介手数料・印紙税・手取り額をプラン別にリアルタイム計算
- 売主様提示用のPDF出力(A4・印刷)に対応
- 担当者設定はブラウザに保存(localStorage)

## 使い方(社内)

ブラウザで GitHub Pages のURLを開くだけで利用できます。
担当者情報は初回に「担当者設定を保存」を押すと、次回から自動入力されます。

> お客様名・価格などの入力内容はファイルに保存されず、ネット上には残りません。
